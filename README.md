# 🎭 Improv Lab - Лендинг сайт

Современный лендинг для команды импровизаторов Improv Lab, созданный на основе информации из Telegram бота.

## ✨ Особенности

- **Современный дизайн** с градиентами и анимациями
- **Полностью адаптивный** для всех устройств
- **Интерактивные элементы** с плавными переходами
- **Интеграция с соцсетями** и Telegram ботом
- **Оптимизированная производительность**
- **SEO-дружественная структура**

## 🚀 Быстрый старт

1. **Клонируйте или скачайте** файлы проекта
2. **Откройте** `index.html` в браузере
3. **Наслаждайтесь** красивым лендингом!

## 📁 Структура проекта

```
improvlab-web/
├── index.html          # Основная HTML страница
├── styles.css          # CSS стили и анимации
├── script.js           # JavaScript функциональность
└── README.md           # Документация
```

## 🔧 Настройка

### Изменение контента:
1. Отредактируйте `index.html` для изменения текста
2. Обновите `styles.css` для изменения стилей
3. Модифицируйте `script.js` для изменения поведения

### Добавление новых секций:
1. Добавьте HTML разметку в `index.html`
2. Создайте стили в `styles.css`
3. Добавьте JavaScript логику в `script.js`
